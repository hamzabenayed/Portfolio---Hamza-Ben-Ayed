Project Description

Tawasalna is a mobile application designed for social interactions, e-commerce, and community engagement in residential areas. It allows users to join groups, publish announcements, and make payments via Stripe integration.

Key Features

  • Multi-language support (English, French, Arabic, Spanish)
  • Interest-based groups and discussions
  • Stripe payment integration
  • Interactive surveys and polls

Technologies Used

  • Frontend: React Native, Redux, Expo
  • Backend: Spring Boot, MongoDB
  • Others: Stripe API, Docker, Kubernetes

Challenges & Solutions

Challenge: Implementing real-time chat in groups

Solution: Used WebSockets and optimized database queries for better performance

Role & Contributions

  • Developed the backend API using Spring Boot
  • Implemented the frontend UI in React Native

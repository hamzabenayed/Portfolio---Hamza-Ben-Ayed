Tawasalna - Social Networking and Commerce App

A mobile platform that combines social interaction, community groups, messaging, and monetization workflows for users in Gulf markets.

Mobile Commerce Real-time Messaging AI Assistance Secure Payments

Project Overview

Tawasalna was built as a unified community platform where users can engage through social content, private messaging, and interest groups while accessing commerce features in the same mobile experience.

Key Capabilities

  • Secure onboarding flows with authentication and account lifecycle management.
  • Dynamic feed with rich content publishing and engagement actions.
  • WebSocket-based private messaging and notification channels.
  • Group management features with configurable privacy controls.
  • Payment integration for premium feature access.
  • AI assistant integration to support user help and discovery.

Architecture Highlights

Mobile Experience

React Native frontend designed for responsive community interactions.

Core Services

Spring Boot APIs for identity, content, and application workflows.

Realtime Layer

WebSockets and push notifications to deliver low-latency communication.

Intelligence and Payments

Dialogflow-based assistant and Stripe integration for monetization.

Challenges and Solutions

  • Balanced feature richness with responsiveness by optimizing API and socket usage.
  • Strengthened data protection through token-based access control and secured flows.
  • Improved multi-language usability with dedicated localization implementation.
  • Stabilized payment journeys with secure transaction handling and validation checks.

Role and Contributions

  • Led end-to-end implementation across backend services and mobile delivery.
  • Designed security architecture and access-control patterns.
  • Implemented messaging and notification systems for live user interactions.
  • Integrated payment and AI assistant modules into the product workflow.
  • Drove quality testing and release-readiness improvements.
